# 📌 Project Overview

This project analyzes employee attrition using HR data to identify workforce trends, monitor employee turnover, and support data-driven HR decision-making.

The project follows a complete **end-to-end data analytics workflow**, beginning with data cleaning and exploratory analysis using Python, followed by SQL-based business analysis and an interactive Power BI dashboard.

---

# 🎯 Business Problem

Employee attrition is one of the biggest challenges for organizations. High employee turnover increases recruitment costs, affects productivity, and results in the loss of experienced employees.

The objective of this project is to help HR teams:

- Monitor employee attrition
- Identify high-risk departments
- Analyze workforce demographics
- Improve employee retention
- Support strategic HR decisions
