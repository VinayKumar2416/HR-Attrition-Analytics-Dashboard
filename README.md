# HR Attrition Analytics Dashboard

## Project Overview

This project analyzes employee attrition using Python, SQL, and Power BI. The goal is to identify workforce trends, understand the factors influencing employee attrition, and support data-driven HR decision-making.

---

## Business Problem

Employee attrition increases recruitment costs, reduces productivity, and impacts business performance. This project helps HR teams identify attrition patterns and improve employee retention through data analysis.

---

## Tools Used

- Python
- SQL
- PostgreSQL
- Power BI
- DAX
- Microsoft Excel

---

## Dataset

- **Dataset:** IBM HR Analytics Employee Attrition
- **Total Records:** 1,470
- **Total Columns:** 31

---

## Project Workflow

1. Data Cleaning using Python
2. Data Analysis using SQL
3. Dashboard Development using Power BI
4. Business Insights and Recommendations
